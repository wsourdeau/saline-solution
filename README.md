Saline Solution


### Introduction

This project descends from a school project that I wrote in 1994 in chemistry
class in QuickBasic. Although it is meant to be useful and functional, it also
enables me to experiment with various languages and TUI toolkits.

### Functional Purpose

The program calculates chemical properties of salt solutions in water, specifically:

- pH and pOH values
- Hydronium [H₃O⁺] and hydroxide [OH⁻] ion concentrations
- Identification of the dominant reaction (acid hydrolysis, base hydrolysis, salt hydrolysis, or water autoionization)
- Hydrolysis degree (x)
- Remaining ion concentrations after hydrolysis

### Target Users

- Chemistry students
- Anyone needing quick salt solution pH calculations
